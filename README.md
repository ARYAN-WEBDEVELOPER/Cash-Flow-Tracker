📊 Cash Flow Tracker

A responsive and interactive Cash Flow Tracker built using HTML, CSS, and JavaScript. This project allows users to manage their salary and expenses, track balance in real-time, and visualize financial data with charts.

🌐 Live Demo 👉 https://aryan-webdeveloper.github.io/Cash-flow-tracker/

📌 Features 💰 Add and manage salary 🧾 Add/Delete expenses dynamically 📉 Real-time balance calculation 💾 Data persistence using LocalStorage 💱 Currency conversion (INR, USD, EUR) 📊 Pie chart visualization (Chart.js) 📄 Download expense report as PDF (jsPDF) 🔔 Smart notifications (toast system) ⚠️ Budget warning when balance < 10% 📱 Fully responsive modern UI

🛠️ Tech Stack HTML5 CSS3 (Responsive Design) JavaScript (DOM Manipulation) Chart.js jsPDF LocalStorage API
